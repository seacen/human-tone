# 去 AI 味 · 极简规则（中文）

**原则：只做减法。** 一处表达删掉后意思不丢，就删；删不掉才改短、还原实义动词、换地道说法。拿不准就留着。单次出现不算病，盯的是短段里的成堆堆叠、脱离内容、删掉不丢意的。以下按规律排列，每条先诊断、再改法，例子只举一个，不是穷举清单。

1. **空洞评价与拔高 → 删。** 不带信息的定性一律去掉：值得注意的是 / 不难发现 / 无疑 / 极大地 / 具有重要意义。
   _例："这一举措无疑具有里程碑式的意义" → "这一步改变了 X"（说清改了什么，或直接删）。_
2. **无据的绝对化 → 降级或删。** 没有依据的"最 / 唯一 / 从来 / 必然 / 彻底"别用，除非手里真有据。
   _例："这是史上最好的方案" → "目前看这个方案更省事"。_
3. **对仗骨架 → 拆。** "不是 X，而是 Y""与其说…不如说…""既是…也是…"多是用对仗撑气势，直接说结论。
   _例："这不是终点，而是起点" → "这只是开始"。_
4. **三段排比惯性 → 删到够。** 三连、四连排比里，删一项不丢意就删，别为凑节奏留。
5. **叙述者导游 → 删。** 领路词去掉，直接讲事：让我们一起 / 接下来 / 综上所述 / 首先其次最后。
6. **含糊对冲 → 删。** 不承担判断的模糊限定去掉：在某种程度上 / 或许可以说 / 一般而言 / 某种意义上。有实义（真表示不确定）才留。
7. **万能动词空转 → 还原实义动词。** "进行 / 作出 / 加以 / 予以 + 名词"换成一个动词。
   _例："对问题进行分析" → "分析问题"；"作出决定" → "决定"。_
8. **的的不休、被字滥用、名词化 → 拆开、改主动、还原动词。** 一句连用三个以上"的"就拆；"被…所…"改主动；"对…的处理 / …的分析"还原成动词。
   _例："受到了广泛的欢迎" → "很受欢迎"；"一个能够有效解决用户痛点的创新性的方案" → "一个能解决用户痛点的新方案"。_
9. **商务黑话与装腔大词 → 直说做什么。** 赋能 / 抓手 / 闭环 / 落地 / 对齐 / 颗粒度 / 心智 / 护城河 / 组合拳，换成具体动作或结果。
   _例："赋能团队" → "帮团队把 X 做起来"。_
10. **翻译腔（逐字直译）→ 换地道说法。** 反译回英文才通顺的词一律换。
    _例："在一天结束时" → "说到底"；"移动这根针" → "带来实质改变"；"低垂的果实" → "现成的机会"。_
11. **拟物化死隐喻 → 直说那层关系。** 别把抽象事物当植物去"长在 / 长进 / 长出 / 长成"，也别当实物去"托住 / 接住 / 兜住 / 撑住 / 承接"。去掉拟物动词，把关系说白。
    _例："信任长在日常协作里" → "日常协作里慢慢建立信任"；"制度托住了这套流程" → "制度保证了这套流程能跑"。_
12. **比喻失控 → 删多余意象、生造的换直说。** 短程内堆两三个意象就删到剩一个；须解码才懂的生造比喻直接换成一读就懂的话。
    _例（生造，都得改）："焊了几千年的缝""把硬币劈成两半" → 直接说你到底指什么。_
13. **戏剧化短句 → 并回正常句。** 用断裂短句、单句成段制造节奏而不表意的，接回去；为节奏而设的断裂删掉。
    _例："我们赢了。彻底地。毫无悬念。" → "我们赢得很干脆。"_
14. **戏剧化揭露 → 删悬念直接给结论。** "真相是…""说到底…""本质上…""你可能不知道…"式先卖关子再抖包袱，直接讲。
15. **代号泛指 → 直接点名。** 用"这 / 那 / 某种 / 这么个东西"代指本可点名的概念时，把那个东西的名字写出来。
    _例："这里面有某种说不清的东西在起作用" → "起作用的是复购率"。_
16. **冗余范畴壳 → 去壳。** 给本已干净的名词缀"…这种东西 / …这种事 / 作为一种…"，把壳删掉，直接说那个名词。
    _例："公司这种东西，本质上是…" → "公司本质上是…"。_
17. **总结书签 → 删。** 开场"在…的今天 / 时代"，结尾"总之 / 综上 / 希望本文…"，去掉。

**别改（护栏，以下不是 AI 味）：**
- 术语、公认缩写（RPI / SKU / NRR…）、成分名、专有名词、人名昵称。
- 引文、代码、命令、数字、日期、版本号——原样保留。
- 公文、法律、学术、技术文档里的规范表达（"予以处理""进行分析"在这些语体里是规范，不是病）。
- 有意的排比、成语、修辞节奏（演讲、宣言、诗体里刻意为之的）。
- 真人明显的口语碎句、不规整、个人声口——门槛更高，倾向别动。

**一句尺子：** 删完别把文章弄得太平、太匀、像中性说明书，那也是一种失败。宁可少删，不可误伤——抹掉真人的判断、语气和细节，比残留一点 AI 腔更不可逆。
